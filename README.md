# Blinkit Sales & Performance Dashboard

## Project Overview
This project presents an interactive **Power BI dashboard** built to analyze Blinkit grocery sales data. The objective of this analysis is to understand product performance, outlet performance, customer ratings, and overall sales trends. The dashboard transforms raw sales data into meaningful business insights that support data-driven decision making.

## Objectives
- Analyze overall sales performance.
- Identify top performing product categories.
- Understand sales distribution across outlet types and locations.
- Evaluate customer satisfaction through ratings.
- Discover sales trends and key business insights.

## Dataset Overview
The dataset contains information related to Blinkit grocery product sales and outlet details.

### Key Columns
- Item Identifier – Unique product ID  
- Item Type – Product category  
- Item Fat Content – Low Fat or Regular  
- Item Weight – Weight of the product  
- Item Visibility – Product display visibility  
- Outlet Identifier – Unique outlet ID  
- Outlet Establishment Year – Year the outlet started  
- Outlet Size – Small, Medium, or Large  
- Outlet Location Type – Tier 1, Tier 2, Tier 3  
- Outlet Type – Grocery Store / Supermarket  
- Sales – Total sales value  
- Rating – Customer rating for the product  

## Tools & Technologies Used
- **Power BI** – Data visualization and dashboard creation  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
- **Excel** – Data source  

## Data Preparation
Data cleaning and transformation were performed using Power Query in Power BI.

Key steps included:
- Standardizing Item Fat Content values
- Handling missing values
- Correcting data types
- Preparing the dataset for analysis

## Dashboard Features
The Power BI dashboard includes the following visualizations:

### Key Performance Indicators (KPIs)
- Total Sales
- Average Sales
- Total Items
- Average Customer Rating

### Visualizations
- Sales by Product Category
- Sales by Outlet Location
- Sales by Outlet Type
- Sales Trend Over Years
- Top Performing Products
- Average Rating by Product Category
- Sales Distribution by Item Fat Content
- Sales Performance by Outlet Size

## Key Insights
- Snack foods and fruits & vegetables contribute significantly to total sales.
- Supermarket Type1 outlets generate the highest revenue.
- Tier 3 outlet locations show strong demand and contribute a large share of sales.
- Medium sized outlets demonstrate strong performance.
- Customer ratings are generally positive, indicating good customer satisfaction.

## Conclusion
The Blinkit Sales Dashboard provides valuable insights into product performance, outlet efficiency, and customer satisfaction. By analyzing sales data through interactive visualizations, the dashboard helps identify growth opportunities, optimize product offerings, and support strategic business decisions.
